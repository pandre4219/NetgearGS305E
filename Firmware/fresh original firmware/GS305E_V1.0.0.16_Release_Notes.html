Enhancements:

  • You can now use the new NETGEAR Discovery Tool (NDT) or the latest NETGEAR Insight app to discover the switch in the same subnet. The NDT supports UPnP, which is now the preferred method of discovery.

Warnings:

Read and follow these warnings before updating your firmware:

  • As of firmware version 1.0.0.16, the NETGEAR Switch Discovery Protocol (NSDP) that was used by the old NETGEAR Switch Discovery Tool (NSDT) is no longer supported. Instead, now use the NDT or the latest NETGEAR Insight app to discover the switch in the same subnet.

Download Link: https://www.downloads.netgear.com/files/GDC/GS305E/GS305E_V1.0.0.16.zip

Firmware Update Instructions:

To update your product’s firmware, follow the instructions in your product’s user manual. To find your user manual, visit https://www.netgear.com/support/, enter your model number in the search box, and click the Documentation button on the product page.
